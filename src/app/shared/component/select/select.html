Select Component

The Select component allows users to choose an option from a dropdown list. It is optimized for mobile use, providing a touch-friendly interface and adaptive design to enhance user experience on smaller screens.

Known Issue

Overlapping Option List

When the Select component is opened, if the option list contains long options that exceed the width of the container element, the list overlaps with parent elements, resulting in truncated or improperly displayed options.

Default

Current Behavior

Long option causes overflow regardless the width of the select element

Custom

Expected Behavior

Options display properly within container bounds

Technical Fix
The fix involves customizing the component's overlay panel and option label styles using PrimeNG's pt property. By setting the overlay panel's root pcOverlay to full width and applying word-break and whitespace styles to the option labels, the options are forced to wrap within the container, preventing overflow and ensuring proper display on mobile devices.
